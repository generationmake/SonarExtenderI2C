# SonarExtenderI2C
Connect up to four Ultrasonic sensor HC-SR04 via I2C to an Arduino or Raspberry Pi

Upper 8 bit
Bit 7: trig US3
Bit 6: trig US2
Bit 5: trig US1
Bit 4: trig US0
Bit 3: status Echo
Bit 2: decode Sense 2
Bit 1: decode Sense 1
Bit 0: decode Sense 0

lower 8 bit
Counter register = proximity
